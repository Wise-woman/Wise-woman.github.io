Git ja GitHub

- Gitin on luonnut Helsingin yliopiston tietojenkäsittelytieteen laitoksella 90-luvulla opiskellut Linus Torvalds.

- Torvalds tunnetaan luultavasti paremmin Linux-kernelin kehittäjänä.

- GitHub on myöhemmin kehitetty palvelu, jonka avulla voidaan säilyttää ja julkaista Git-projekteja.

- GitHub tekee yhteistyöstä helpompaa gitin avulla.

- GitHub on alusta, joka voi sisältää koodivarastoja pilvipohjaisessa tallennustilassa.

- Edellä mainittu mahdollistaa reaaliaikaisen ja samanaikaisen kehittämisen.

- GitHub tarjoaa myös kehittäjille erinomaisen verkostoitumismahdollisuuden.
